## StableSafe contracts

Using [Foundry](https://book.getfoundry.sh)

## Usage

### Build

```shell
forge build
```

### Test

```shell
forge test
```

### Deploy

```shell
forge script script/Counter.s.sol:CounterScript --rpc-url <your_rpc_url> --private-key <your_private_key>
```

### Help

```shell
forge --help
anvil --help
cast --help
```
